# Harness Legacy Archive

This directory keeps historical material that has already been folded into
active Harness docs or runbooks.

Use order:

1. Start with [../../index.md](../../index.md).
2. Follow `current_truth` from
   [../../legacy-conversion-index.md](../../legacy-conversion-index.md) or a completed
   execution-plan stub.
3. Open files here only for provenance, incident evidence, or commit-message
   reconstruction.

Archived bodies are not authoritative runbooks. If a fact here still matters,
move that fact into an active doc and leave only the evidence trail here.
