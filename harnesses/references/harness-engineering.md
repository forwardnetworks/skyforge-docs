# Harness Engineering Reference

Source: https://openai.com/index/harness-engineering/

Skyforge adopts these ideas from the article:

- `AGENTS.md` is a table of contents, not an encyclopedia.
- Repository-local docs are the system of record for agent knowledge.
- Execution plans are first-class versioned artifacts.
- Architecture and taste should be enforced mechanically where possible.
- Recurring doc gardening and quality scoring prevent agent-generated entropy.

This file is a compact pointer, not a replacement for the source article.
